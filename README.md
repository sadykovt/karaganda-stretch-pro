# Потолок-КРГ

Сайт компании по установке натяжных потолков в Караганде.

## Технологии

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Разработка

1. Установите зависимости: `npm install`
2. Запустите dev-сервер: `npm run dev` (http://localhost:8080)
3. Сборка: `npm run build`
4. Предпросмотр: `npm run preview`
